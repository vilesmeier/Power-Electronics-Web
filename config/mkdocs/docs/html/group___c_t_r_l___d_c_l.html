◆ DCL_runClamp_C1()

int16_t DCL_runClamp_C1 ( float32_t * data,
float32_t Umax,
float32_t Umin )
extern

Saturates a control variable and returns 1 if either limit is exceeded. Implemented as an external assembly module.

Can be used to saturate a pre-computed Direct Form 2 controller. If the immediate result is in range it can be used, otherwise it can be clamped and the next partial pre-computation skipped. An example of use with a pre-computed DF22 controller follows:

uk = DCL_runDF22_C2(&arma2, rk); // immediate result from pre-computed controller
i = DCL_runClamp_C1(&uk, 1.0f, -1.0f); // clamp immediate result to +/-1.0
// ...use uk here...
if (0 == i) // if immediate result is in range...
{
DCL_runDF22_C3(&arma2, rk, uk); // ...pre-compute the next partial result
}
