## 平台支持与基本功能
### 软件支持
1. 支持C2000 Gen3 系列器件, 部分功能**不支持STM32** 
2. 支持同时使用寄存器 驱动库 syscfg
3. 支持CCS12 CCS20
4. 支持*Doxygen*生成API文档
5. 提供环路控制层/硬件驱动层代码, 应用层代码框架, 完整题解、拓扑控制环路的代码工程

### 硬件支持
1. 立创EDA-pro-V3.x
2. Altium Designer 2024
3. 支持常用单板功率级, 采样模块, 通用DSP主板, 控制卡

### 仿真支持
1. Matlab 2024b    数字环路仿真
2. Multisim 14     模拟环路仿真
3. Plece 4.7       代码功能验证
