void Device_init (void)
 Function to initialize the device. Primarily initializes system control to a known state by disabling the watchdog, setting up the SYSCLKOUT frequency, and enabling the clocks to the peripherals. The function also configures the GPIO pins 22 and 23 in digital mode. To configure these pins as analog pins, use the function GPIO_setAnalogMode
