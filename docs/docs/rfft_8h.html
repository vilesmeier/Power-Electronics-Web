◆ FFT_MODE

enum FFT_MODE

定义RFFT频谱分析模式

枚举值
FFT_MODE_NULL 

初始化后未配置, 尚未指定FFT分析模式

FFT_GRID_LOCK 

delta_f == fgrid, 直接处理幅度谱内每一个元素

FFT_PEAK_FIND 

delta_f != fgrid, 只处理幅度谱内的峰值点
