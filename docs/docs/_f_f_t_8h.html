注解
  • 本文件只支持FPU32, 无FPU32的设备不建议使用
  • 有VCU的设备建议使用支持VCU的版本以获得更高的效率
  • RFFT_Analyser 结构体内的数组均需要对齐
  • TI 官方库, 请参考TI 官方的DSP库用户手册
  • System Config工具支持FFT结构体生成, 相关生成文件在 /dsp_lib 文件夹内
