详细描述

FPU register space

注解
  • The registers are mapped to data space only (like peripherals on C28x devices). Program space data reads or writes are ignored. debug reads/writes work
  • All registers are reset to zero on system reset (SYSRSn).
  • The RB register is part of the C28-CPI core but is made visible (read only to CPU) in the memory map. Refer to "CPI Specification" for more details.
