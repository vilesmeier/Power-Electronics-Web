# 数据手册
## 简介
本文件夹内包括芯片的用户手册以及实验室仪器的用户说明书

## 文件结构
```
├── DataSheet           数据手册
│   ├── DSP             主控相关手册，按主控区分
│   │   ├── F28P650
│   │   ├── F28335
│   │   ├── F28379D
│   │   └── F280049C
│   ├── 仪器            仪器说明书
│   └── 芯片            芯片数据手册
```

## /DSP
- C2000芯片的用户手册、数据手册、应用手册等TI官方文件
- 部分核心板的资料
- TI控制卡开源资料
- F28335/F28379D/F280049C/F28P650DK

## /仪器
 - 实验室现有的部分仪器说明书
 - 实验台:示波器/信号源/万用表/直流源
 - 会议室：三相电表/单相功率分析仪/三相电表（三相功分青春版）/LCR电桥

## /芯片
 - 硬件所用芯片的数据手册
 - 串口屏及其开发手册
